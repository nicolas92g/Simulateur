Detailed Description

this define a 2d texture that you can draw with a position, a scale and rotation the 0, 0 position is on the bottom left corner of the window and the texture position point is on the middle this Object can't resize up a texture so the scale of the object is limited by the texture resolution the scale is in pixel the position is in pixel the rotation is in degrees you can set a layer which allow to say which one will be draw on top of which one
